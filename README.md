# TypeScript Mastery: The Complete Reference Guide

**Description:** This document serves as a comprehensive, zero-to-hero reference manual for TypeScript. It is designed to bridge the gap between the unpredictable nature of plain JavaScript and the strict, type-safe environments of compiled languages like C++. 

**Purpose:**
The core objective of these notes is to master how to catch logic errors, enforce data structures, and build robust, crash-resistant web applications before the code ever runs in a browser or server—especially when structuring complex, large-scale projects in frameworks like Next.js.

**Scope:**
This guide covers 32 essential topics divided into five phases:
1.  **The Absolute Foundations:** Transpilation, inference, and primitive types.
2.  **Core Structures:** Interfaces, type aliases, and function typing.
3.  **Intermediate Type Manipulation:** Unions, narrowing, and type guards.
4.  **Advanced Typing:** Generics, utility types, and conditional logic.
5.  **Architecture & Tooling:** Project configuration and strictness rules.

*Note to self: Trust the compiler. Types are erased at runtime, so external data must still be validated, but internal logic should be flawlessly typed.*
